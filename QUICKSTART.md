# Zendesk Claude Code Workshop - Quick Start

## 🚀 One-Line Setup (Copy & Paste This!)

Open **Terminal** on your Mac and paste this command:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/mattheqd/zendesk-ai-onboarding/main/setup-wizard.sh)
```

Press **Enter** and follow the wizard!

---

## Before You Start

Make sure you have:

1. ✅ **GlobalProtect VPN** connected
2. ✅ **Zendesk email** (for GitHub access)
3. ✅ **AI Gateway token** ready (the wizard will help you get this)

---

## During Setup

The wizard will:
1. Check your VPN connection
2. Verify GitHub access (and tell you how to get it if needed)
3. Help you get your AI Gateway token
4. Install all required tools (~10 minutes)
5. Set up Claude Code

---

## GitHub Access (Important!)

**If you're not in the Zendesk GitHub org yet:**

Before running the setup, ping **#it-help** on Slack:

> "Hi! I need to be added to the Zendesk GitHub organization for the Claude Code workshop. My GitHub username is [YOUR_GITHUB_USERNAME]"

This takes ~15 minutes if you ask today, 2 days if you wait until the workshop!

---

## Troubleshooting

### "Cannot reach ai-gateway.zende.sk"
→ Connect to GlobalProtect VPN first

### "Token doesn't look right"
→ Make sure it starts with `zdai_` and has no extra spaces

### "Not a member of zendesk org"
→ See GitHub Access section above
