Getting Your AI Gateway Token

The wizard will help you with this, but here's the flow:

  • Open https://ai-gateway.zende.sk
  • Log in with your Zendesk credentials
  • Copy your API token (starts with zdai_)
  • Paste it into the wizard when prompted
1

Before You Start

Make sure you have:

  • GlobalProtect VPN connected
  • Zendesk email (for GitHub access)
  • 10 minutes of uninterrupted time
2

During Setup

The wizard will:

  • Check your VPN connection
  • Verify GitHub access (and tell you how to get it if needed)
  • Open browser to get your AI Gateway token
  • Install all required tools automatically
  • Set up Claude Code with your token
3

What Gets Installed

The wizard installs everything you need:

  • Homebrew (Mac package manager)
  • Node.js 20+
  • GitHub CLI
  • VS Code
  • Claude Code

Total size: ~350 MB

❓ Troubleshooting

"Cannot reach ai-gateway.zende.sk"

→ Connect to GlobalProtect VPN first

"Token doesn't look right"

→ Make sure it starts with zdai_ and has no extra spaces

"Not a member of zendesk org"

→ See the GitHub Access section above - ping #it-help on Slack

GitHub authentication fails

→ Choose "GitHub.com", "HTTPS", and "Login with web browser"

Other issues?

→ Contact workshop organizers or ping #it-help
