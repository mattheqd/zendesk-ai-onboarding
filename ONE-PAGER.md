# 🚀 Claude Code Workshop Setup

## Before the Workshop: 10-Minute Setup

### Step 1️⃣: Connect VPN
Open **GlobalProtect** and connect

### Step 2️⃣: Open Terminal
Find it in Applications → Utilities → Terminal

### Step 3️⃣: Copy & Paste This
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/mattheqd/zendesk-ai-onboarding/main/setup-wizard.sh)
```

### Step 4️⃣: Follow the Wizard
It will guide you through everything!

## 🔑 You'll Need Your AI Gateway Token

The wizard will open a browser for you, but here's the flow:

1. Go to https://ai-gateway.zende.sk
2. Log in with Zendesk credentials
3. Copy your token (starts with `zdai_`)
4. Paste into the wizard

The wizard installs everything you need:

- ✓ Homebrew (Mac package manager)

- ✓ GitHub CLI

- ✓ Claude Code
